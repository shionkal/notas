Un poco de historia

El nacimiento de la web se da el 23 de agosto de 1991 en el CERN (Conseil Européen pour la Recherche Nucléaire) cuando por primera vez en la historia usuarios externos al CERN pudieron empezar a acceder a su última creación. Su desarrolló se dio años antes por el científico informático Tim Berners-Lee, y se trataba de una red con la cual poder conectar varios documentos a través de hipervínculos, con el objetivo de ayudar a los físicos en la búsqueda de la información.

Para 1993 había 600 sitios, para 1995 la cifra aumentó hasta los 15.000. Hoy, la WWW se ha convertido en el servicio más valorado de Internet. A partir de 2008, había 1500 millones de usuarios de Internet en todo el mundo, 1300 millones de usuarios de correo electrónico, 210 mil millones de correos electrónicos enviados diariamente, 186,7 millones de sitios web y 133 millones de blogs.

El lenguaje de la web: HTML

Un sitio Web está formado por un conjunto de recursos o páginas web conectadas, generalmente asociado a un tema bien definido, que se ponen a disposición de los usuarios de Internet a través de una dirección web URL. Hoy en día, un sitio web representa el medio más moderno de publicación y comunicación. La creación de una página web requiere el uso de un lenguaje específico, llamado HTML (Hyper Text Markup Language). HTML es un lenguaje de “marcado” que se utiliza para la creación, representación y estructuración del contenido de una página web mediante elementos de formato o etiquetas.Estas etiquetas tienen significados bien definidos, como insertar una imagen, crear un enlace o un párrafo, etc., y generalmente se insertan en el texto de un documento antes de que el navegador web las intérprete y muestre. HTML es un estándar publicado por W3C (World Wide Web Consortium)

Sintaxis HTML

El lenguaje HTML fue creado para estructurar y dar sentido al contenido de la página web. Su sintaxis gira esencialmente en torno a tres términos básicos: elemento, etiqueta y atributo.

Elementos

Los elementos se refieren a los identificadores que describen objetos (texto, imagen u objeto multimedia) dentro de una página web, incluida la estructura y el contenido. Los elementos estan formados por una etiqueta de apertura, contenido y una etiqueta de cierre.

Etiquetas

El término “etiqueta” es uno de los conceptos básicos del lenguaje HTML. Los elementos en HTML a menudo se componen de varios conjuntos de etiquetas. Estas etiquetas no se muestran en la página pero es gracias al navegador instalado en el ordenador del usuario que su efecto se mostrará correctamente. Hay dos tipos de etiquetas:

Etiquetas emparejadas: formadas por dos etiquetas, una de apertura y otra de cierre. La etiqueta de apertura está marcada al principio con el signo "<" y al final con el signo “>”. La etiqueta de cierre sigue el mismo principio, pero el nombre del elemento está precedido por el carácter de diagonal "/".

Ejemplo.-
Etiqueta para definir texto importante
<strong> mostrar un texto en negrita</strong >

Etiquetas singulares: Estas etiquetas se utilizan con mayor frecuencia para insertar un elemento proveniente de un recurso externo en en una ubicación bien definida en la página web. Para este tipo de etiqueta, no es necesario marcar el inicio y final de este elemento.

Ejemplo .-
Etiqueta para agregar una imagen
< img src="http:/logo.jpg" alt="Logo"/ >

Nota.- Para no confundir los conceptos de “etiqueta” y “elemento”, consideremos el siguiente ejemplo:
<p> Este elemento contiene texto normal y términos en <b> negritas </b> </p>

Además del texto, el elemento de parrafo <p> contiene otras etiquetas (en este caso, < b > como ejemplo), mientras que la etiqueta <b > solo contiene texto. Notamos que el contenido de un elemento pueden ser otro elemento con sus respectivas etiquetas.

Atributos

Los atributos son propiedades que se utilizan para agregar declaraciones adicionales a un elemento dado (un identificador (id), una fuente (src), etc.). Siempre están incrustados en la etiqueta de apertura y deben estar separados entre sí por al menos un espacio. Cada atributo generalmente acepta un valor entre comillas. Un elemento con atributos puede aceptar la siguiente sintaxis:

< nombre_etiqueta atributo="valor" > Contenido del elemento < /nombre_etiqueta >

Ejemplo.-
< video src="video/vacaciones.mp4" > < /video >

Estructura básica documento html

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title<Document</title>
    </head>
    <body>
    
    </body>>
    </html>
  • <!DOCTYPE html>.- Indica la versión del tipo de documento html, en este caso indica que el documento esta escrito con la version html5, la cual es la versión más reciente y actualmente se recomienda escribir todas las páginas con esta versión.
  • </html>.- Este elemento encierra todo el contenido de la página entera y, a veces, se le conoce como el elemento raíz (root element).
  • <head></head>.- Sirve para agregar los metadatos de la página, como palabras clave (keywords), una descripción de la página que quieres que aparezca en resultados de búsquedas, código CSS para dar estilo al contenido, etc.
  • <meta charset="utf-8"> Este elemento la codificación de caracteres que tu documento usará en utf-8 (8-bit Unicode Transformation Format)
  • <title></title> Título de la página.
  • <body></body> Encierra todo el contenido visible de la página.

Texto con énfasis: < em >

Marcar con énfasis las partes importantes de un texto

Texto importante: <strong>

Resalta con especial énfasis las partes más importantes de un texto.

Ejemplo.-

<p> <em>El dinero</em>es importante pero <strong>la salud</strong> lo es más. </p>

Resultado

El dinero es importante pero la salud lo es más.

Texto resaltado: <mark>

Representa un texto marcado o resaltado como referencia o anotación

Ejemplo.-

<p> El telescopio espacial <mark> Hubble </mark> de la NASA ha sacado muchas imágenes asombrosas. Una de las más memorables es el mosaico del perfil del Sombrero. Con su magnitud relativamente alta y a una distancia de <mark> 28 millones de años luz </mark> de la Tierra, Messier 104, nombre formal de Sombrero, se puede ver fácilmente por un telescopio. </p>

Resultado

El telescopio espacial Hubble de la NASA ha sacado muchas imágenes asombrosas. Una de las más memorables es el mosaico del perfil del Sombrero. Con su magnitud relativamente alta y a una distancia de 28 millones de años luz de la Tierra, Messier 104, nombre formal de Sombrero, se puede ver fácilmente por un telescopio.

Texto de código

Es el apropiado para marcar el código de un programa.

Ejemplo.-

<code>
    <pre>
    h1{
        background-color:blue;
    }
    </pre>
</code>

Resultado

h1{
    background-color:blue;
}

Etiquetas de listas

Listas ordenadas: < ol > ,< li >

Estas etiquetas permiten agregar texto en formato de lista con una prioridad en el orden de aparición de los elementos.

Ejemplo.-

    Equipos con más títulos de champions
    <ol>
        <li>Real Madrid</li>
        <li>Milan</li>
        <li>Liverpool</li>
    </ol>

Resultado

  1. Real Madrid
  2. Milan
  3. Liverpool

Listas desordenadas: < ul > ,< li >

Estas etiquetas permiten agregar texto en formato de lista sin una prioridad en el orden de aparición de los elementos.

Ejemplo.-

    <ul>
        <li>Manzanas</li>
        <li>Peras</li>
        <li>Sandias</li>
    </ul>

Resultado

  • Manzanas
  • Peras
  • Sandias

Tablas

Contenedor de la tabla: < table >

Esta etiqueta encierra todo el contenido de una tabla

Filas: < tr >

Agrega una fila a la tabla

Celdas de encabezado: < th >

Esta etiqueta permite agregar celdas para los encabezados de columna o fila

Datos: < td >

Esta etiqueta permite agregar los diferentes datos dentro de la tabla, deben estar dentro de la etiqueta <tr >

Ejemplo.-

Pais Habitantes Territorio Idioma
México 198.9 Español
Brasil 212.6 8516 Portugues
Canada 38.01 9985 Ingles/Frances

Es posible tambien crear celdas que ocupen más de una columna o más de una fila, para ello se usan los atributos colspan y rowspan. Si se busca que una celda se expanda sobre varias columnas se emplea colspan con el valor del numero de columnas que ocupara dicha celda, es decir si buscamos se expanda 2 columnas se agregara el atributo colspan="2".El atributo rowspan se usa de forma similar, se coloca en la celda que ocupara mas de una fila indicando en el valor el numero de filas.

Etiquetas multimedia

Imágenes

Permite agregar imágenes, debe incluir el atributo obligatorio src cuyo valor corresponde a la ruta donde se encuentra la imágen.

Ejemplo.-

<img src="img/esme.webp" alt="cascada esmeralda, oaxaca">

Resultado

Video

Permite agregar video directamente desde el navegador, agregada en la versión de HTML5

Ejemplo.-

<video src="video/big_buck_bunny.ogv" controls> </video>

Atributo Valor Descripción
src Dirección URL Video a reproducir. Obligatoria si actua como etiqueta contenedora.
poster Dirección URL de la imagen Muestra una imagen a modo de presentación.
autoplay Booleano Comienza a reproducir el video automáticamente.
loop Booleano Vuelve a iniciar el video cuando finaliza su reproducción (bucle).
muted Booleano Establece el video sin sonido (silenciado).
controls Booleano Muestra los controles de reproducción. Por defecto no se muestran.
width tamaño Indica el tamaño de ancho del video.
height tamaño Indica el tamaño de alto del video.

La etiqueta <video> también puede actuar como etiqueta contenedora e incluir varias etiquetas HTML para dotar de mayor compatibilidad, o capacidades adicionales.

Video con alternativas: <source >

Si utilizamos la etiqueta <video> como etiqueta contenedora, podemos incluir etiquetas <source > en su interior para proporcionar formatos alternativos y tener mayor compatibilidad con otros navegadores y navegadores antiguos que no soporten HTML5:

Ejemplo


    <video controls>
        <source src="video/big_buck_bunny.webm" type="video/webm">
        <source src="video/big_buck_bunny.ogv" type="video/ogg">
        <source src="video/big_buck_bunny.mp4" type="video/mp4">
        <img src="imagen.png" alt="Video no soportado" />
        Lo lamento; su navegador no soporta video HTML5.
    </video>
